# Chillax Rain Themes

This fork is now focused on Rain for iOS.

## Themes

- `rain/manifest.json`
  - Base Chillax Rain port
- `rain/chillax-glass-sakura.json`
  - Chillax Glass Sakura variant with the Sakura Miku / My Melody background

## Install In Rain

In Rain, open `Settings > Themes > Install from URL` and paste one of these:

- `https://raw.githubusercontent.com/Matthewcappelli/Chillax-forked-/main/rain/manifest.json`
- `https://raw.githubusercontent.com/Matthewcappelli/Chillax-forked-/main/rain/chillax-glass-sakura.json`

## Repo Layout

- `rain/`
  - Rain theme manifests
- `Wallpapers/`
  - Background images used by the themes

## Notes

- The old desktop BetterDiscord/Vencord CSS files were removed to keep the fork clean and Rain-specific.
- The Sakura theme includes a dedicated keyboard color so the iPhone keyboard matches the theme better.
